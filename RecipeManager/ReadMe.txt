Recipe Manager
==============
A .NET project to manage recipes

==============
Overview
==============
This project provides a simple implementation of a Recipe class with properties for the name, ingredients, instructions, meal type, and category of a recipe.

==============
Class Definition
==============
The Recipe class has the following properties:

Name: the name of the recipe as a string
Ingredients: the ingredients required for the recipe as a string
Instructions: the instructions to make the recipe as a string
MealType: the meal type of the recipe as a string (e.g. breakfast, lunch, dinner)
Category: the category of the recipe as a string (e.g. vegetarian, vegan, gluten-free)
The Recipe class has a constructor that takes parameters for the name, ingredients, instructions, meal type, and category and sets the corresponding properties.
